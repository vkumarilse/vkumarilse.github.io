CC5-Accessing data: Scraper and API

API Task: Chart 5A uses live data from the World Bank API to visualize and compare Renewable Energy Consumption of China, India and US with the World. The API structure follows: https://api.worldbank.org/v2/country/{code}/indicator/{indicator}?format=json. Full URL used: https://api.worldbank.org/v2/country/IN;1W;CN;US/indicator/EG.FEC.RNEW.ZS?date=1990:2023&format=json&per_page=500

Scraper Task: Chart 5B displays Nifty 50 stock data scraped from NSE India's website using Python. The chart displays stock performance metrics including price movements and trading volumes (for one day). The scraping process required bypassing anti-bot measures through session management and proper HTTP headers. Data was cleaned and transformed from nested JSON into tidy format, this meant that data extraction was feasible for smaller timeframe. Challenges included anti-bot headers, nested JSON responses, rate limiting, and AJAX-loaded content. Solutions: Python requests with session cookies, custom user-agent headers, and JSON parsing for nested structures.

CC10-Advanced Analysis and Machine Learning

Cluster Analysis: Indian states are grouped into distinct clusters based on their banking sector growth patterns (Credit and Deposit CAGR).

Method: Unsupervised learning using K-Means clustering in sklearn. Input features (X matrix) included standardised Credit CAGR and Deposit CAGR for each state (2016-2024). Optimal cluster count (k=4) was determined using elbow method analysis.

Findings: Four distinct clusters emerged: (1) High credit growth states averaging 16.4% Credit CAGR (Blue); (2) Balanced high-growth states at 15.7% credit/15.8% deposit (Green); (3) Moderate growth states (Yellow); (4) Low-growth states at 7.9% credit/9.2% deposit (Orange). Northeastern states dominated the high-growth clusters.

This composite visualization presents the K-Means clustering results for Indian state banking data. The main scatter plot displays all 36 states/UTs positioned by their Credit CAGR (x-axis) and Deposit CAGR (y-axis), with four distinct cluster colours revealing growth pattern groupings. Cluster 1 captures high-credit-growth states like Manipur (21.3%) and Sikkim (17.6%), primarily northeastern regions experiencing rapid financial deepening. Cluster 4 contains slower-growth states including West Bengal and Punjab, and also includes states which already have sufficient credit deposit penetration like Maharashtra and Delhi.The histogram panel below shows the distribution of states based on credit and deposit CAGR individually, the colour scale is based on the clusters assigned i.e High, Balanced, Moderate and Low growth. As this chart shows a composite chart I have kept the count of charts for this section to one.
